<h1 align="center">Getting Started</h1>

### Usage

1. Download last release.
2. Run `Launcher.exe`.
3. Select `Client-Win64-Shipping.exe`.
4. After opening the game, press `INSERT` to open the GUI.

<h1 align="center">Features</h1>

#### Player

- Hit Multiplier
- Character Speed
- God Mode

#### World

- Dump Enemies
- Game Speed

#### Misc

- FPS Indicator
- FPS Unlocker
- Load Config
- Save Config
